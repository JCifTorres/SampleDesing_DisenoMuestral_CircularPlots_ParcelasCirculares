# SampleDesing_DisenoMuestral_CircularPlots_ParcelasCirculares
Script en ArcPy para generar un diseño de muestreo sistemático, estratificado y aleatorio: crea una grilla de UM, identifica el LULC dominante, selecciona UMs por estrato, genera puntos centrales y desplazados, y produce parcelas circulares listas para trabajo de campo.
Puede usar y citar este script así: Torres-Celi, J. (2025). Script ArcPy (ArcGis Pro) para el diseño muestreal sistemático, estratificado y aleatorio para parcelas de tipo circulares (versión 1.0). (Script - Código Fuente). GitHub.
