# SampleDesing_DisenoMuestral_CircularPlots_ParcelasCirculares
Script en ArcPy para generar un diseño de muestreo sistemático, estratificado y aleatorio: crea una grilla de UM, identifica el LULC dominante, selecciona UMs por estrato, genera puntos centrales y desplazados, y produce parcelas circulares listas para trabajo de campo.
Puede usar y citar este script así: Torres-Celi, J. (2025). Script ArcPy (ArcGis Pro) para el diseño muestreal sistemático, estratificado y aleatorio para parcelas de tipo circulares (versión 1.0). (Script - Código Fuente). GitHub. https://github.com/JCifTorres/SampleDesing_DisenoMuestral_CircularPlots_ParcelasCirculares.git 

# SampleDesing_CircularPlots_ParcelasCirculares
ArcPy script to generate a systematic, stratified, and random sampling design: creates a UM grid, identifies the dominant LULC, selects UMs by stratum, generates center and offset points, and produces circular plots ready for fieldwork.
You can use and cite this script as follows: Torres-Celi, J. (2025).
